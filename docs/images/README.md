## Capture quick guide

1. Launch the app, size window to **1280×800**
2. Use Light theme for default captures, Dark for one hero shot
3. Capture with Win+Shift+S (built-in) or ShareX
4. Crop window chrome (optional) and compress with [Squoosh](https://squoosh.app) to <150 KB each
5. Record GIF with [ScreenToGif](https://www.screentogif.com/) at 15 FPS, trim and compress to <2 MB
